Block Elements

This section covers various aspects of HTML block elements. Each element type is described along with its tags, a sample script, and the resultant output.

Div Element

Description: The div element is a generic container for flow content.

Example:

<div>This is a div element.</div>

Output:

This is a div element.

Paragraph Element

Description: The p element represents a paragraph.

Example:

<p>This is a paragraph.</p>

Output:

This is a paragraph.

Heading Elements

Description: The h1 to h6 elements represent six levels of headings.

Example:

<h1>Heading 1</h1>

Output:

Heading 1

List Elements

Description: The ul and ol elements define unordered and ordered lists, respectively.

Example:

<ul><li>Item 1</li><li>Item 2</li></ul>

Output:

  • Item 1
  • Item 2

Blockquote Element

Description: The blockquote element represents a block of text that is a quotation from another source.

Example:

<blockquote>This is a quote.</blockquote>

Output:

This is a quote.
